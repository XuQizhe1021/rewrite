## SEO/内容垃圾过滤器（v1.0）页面与交互设计

### Popup
- 目的：快速触发一次摘要/步骤/对比分析。
- 组件：模式选择、输出风格选择、呈现方式选择（侧边栏/悬浮层）、一键打开 Options。
- 状态：空态、生成中（流式）、完成、错误（可重试）。

### Side Panel
- 目的：长时间停留、可滚动查看生成内容，适合阅读场景。
- 组件：标题、模式/风格显示、Markdown 渲染区、复制/重试/反馈、插入 TL;DR 开关。
- 行为：生成时自动滚动到末尾，用户滚动后不强制跟随。

### Options
- 目的：配置 API Key/Provider/Model、温度、默认风格、默认模式、自动触发白名单。
- 安全提示：Key 不会写入页面脚本/不会上传到开发者服务器。
- 自动模式：添加域名时触发授权提示，显示当前已授权域名列表与撤销入口。

### 页面内悬浮层
- 触发：Popup 选择“悬浮层”或右键菜单触发。
- 组件：可拖拽/可折叠卡片，Markdown 渲染区、复制/重试/关闭。

### TL;DR 顶部条
- 触发：用户在 Side Panel 或 Popup 中选择“插入 TL;DR”。
- 展示：淡黄色背景，一句话摘要 ≤30字，带关闭按钮。
